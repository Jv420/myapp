# DynathiStore

Verkoopbaar Minecraft webshop-pakket voor BuiltByBit.

## Onderdelen

- `plugin/` – Paper/Spigot plugin voor automatische levering
- `webshop/` – Webshop en checkout
- `license-server/` – Licentievalidatie en activaties
- `docs/` – Installatie- en BuiltByBit-documentatie

## Geplande functies

- Automatische levering van ranks, crate keys, coins, kits en consolecommando's
- LuckPerms-integratie
- Stripe/iDEAL checkout
- Discord webhookmeldingen
- Beveiligde webhookvalidatie
- Licenties per klant, domein en Minecraft-server
- Beheerdersdashboard

> Zet nooit geheime sleutels, Stripe-secrets, databasewachtwoorden of private API-tokens in deze publieke repository.
